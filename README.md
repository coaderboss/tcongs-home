A high-end, highly interactive, and fully responsive homepage redesign built for the **Tcongs Infotech Internship Assignment**. This project focuses on delivering a "Silicon Valley" tier user experience with butter-smooth animations, glassmorphism UI, and robust modern web technologies.

---

## 🔗 Live Demo
**Check out the live project here:**  https://tcongs-home.vercel.app

---

## ✨ Key Features & Upgrades

- **Deep Ambient Hero Section:** Features centralized radial gradients, slow-breathing animated glows, and dynamic text reveals that create an immersive first impression.
- **Interactive Services Grid:** Replaced static boxes with premium glassmorphic cards, featuring hover-lift physics and sleek SVG iconography.
- **Scroll-Linked Process Timeline:** A dynamic vertical timeline that draws itself as the user scrolls, combined with sticky headers and staggered content reveal.
- **Glassmorphic Contact Form:** A beautifully frosted contact section with staggered input reveals, glowing focus states, and a shimmering submit button.
- **Awwwards-Level Animations:** Powered by `framer-motion`, featuring word-by-word text masking, spring physics, and scroll-triggered cascade effects.
- **Flawless Responsiveness:** Mobile-first approach ensuring pixel-perfect scaling across phones, tablets, and 4K desktop displays without breaking layouts or introducing dead spaces.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Deployment:** [Vercel](https://vercel.com/)
- **Icons:** Custom Inline SVGs
